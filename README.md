# Mobi Bro Landing

Готовый статический лендинг для сервиса `Mobi Bro`.

## Файлы для публикации

- `index.html`
- `styles.css`
- `logo.svg`
- `favicon.svg`
- `site.webmanifest`
- `robots.txt`

## Как открыть локально

Открой `index.html` в браузере.

## Как загрузить на хостинг

1. Зайди в панель хостинга.
2. Открой папку сайта, обычно это `public_html`, `www` или корень домена.
3. Загрузи туда все файлы из этой папки.
4. Убедись, что `index.html` лежит в корне сайта.
